# Customer Churn Prediction Project
## Project Overview

This project predicts whether a customer will leave (churn) or stay with a company based on basic customer details.
It uses Machine Learning and a Streamlit web app to make real-time predictions.

## Technologies Used

Python

Machine Learning

Streamlit

NumPy

Joblib

## Features

User-friendly web interface

Takes customer input values

Predicts Yes / No for churn

Fast and simple prediction

## Input Parameters

The model takes the following inputs:

Age

Gender

Tenure

Monthly Charge

## How the Project Works

User enters customer details

Data is scaled using a saved scaler

Trained ML model predicts churn

Result is shown as Yes (Churn) or No (No Churn)

## Project Objective

The main goal of this project is to help businesses identify customers who may leave so they can take action to retain them.
